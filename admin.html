🔊 Voice announcements

Spoken aloud on the Queue Board screen when a token is called. Uses the device's built-in text-to-speech engine — no internet required. Configure languages and templates below, then open the Board screen to verify the speaker is working.

Important: each language requires a TTS (text-to-speech) voice to be installed on the device running the Board screen. If a language shows ⚠ on the Board, install that language's TTS voice in the device's Settings → Accessibility → Text-to-speech (Android) or Settings → General → Accessibility → Spoken Content (iOS/iPad).

Variables: {token} = token number   {room} = room   {doctor} = doctor name. Languages play in order when a token is called.
